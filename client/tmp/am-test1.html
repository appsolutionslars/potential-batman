Change Management

    
       
 

appSolute Management is a Change Management application with modules for call logging and knowledge sharing, product management, client and project management, as well as other modules for management within service and development organizations.

For customers using Roundtable TSMS, the application also has integration points with CallDesk and Roundtable tasks providing a link between the management system and development environments.

appSolute Management is developed with OpenEdge from Progress Software. The application uses the Progress AppServer technology on the back-end and the Progress WebClient to provide a rich Windows user interface. It is therefore well suited for use in distributed environments wih users in multiple locations accessing the same backend server processes.
